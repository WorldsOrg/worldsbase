# Worldsbase Unreal Engine Plugin
This Unreal Engine plugin provides a connection to Worldsbase. With it you can read/write to a Worldsbase database and call other APIs.

# Installation
1. Place `wgs` folder into `Plugins` directory <br/><br/> <img src="/images/unreal/wgs-directory.png" width="700"/> <br/><br/> 
# Example 1: Add Data to Table
In this example, we will add data to a table using the `Insert Data` node. Skip to the end of this section to see the full blueprint example.

1. Right click and search for wgs subsystem node <br/><br/> <img src="/images/unreal/subsystem.png" width="500"/> <br/>
2. Drag from subsystem node and search for desired WGS function. In this example, we will use the Insert Data function. <br/><br/> <img src="/images/unreal/insert-data.png" width="500"/> <br/>
3. Fill in the table name <br/><br/> <img src="/images/unreal/table-name.png" width="200"/> <br/>
4. Drag from Data Rows and select the `Make Array` node <br/><br/> <img src="/images/unreal/make-array.png" width="400"/> <br/>
5. Add a pin to the Make Array node for each column you want to edit in your table
6. Drag from each pin and select `Make DataRow` <br/><br/> <img src="/images/unreal/make-data-row.png" width="400"/> ![]() <br/>
7. Fill out column name and value for each data row <br/><br/> <img src="/images/unreal/fill-data-rows.png" width="400"/> <br/>
8. Drag from Insert Data node execution pin and search for `Bind Event to On Http Request Completed`. This will execute once our backend call is complete. <br/><br/>  <img src="/images/unreal/http-complete-bind.png" width="600"/> <br/>
9. Make the subsystem the target for the Bind Event node <br/><br/> <img src="/images/unreal/http-complete-bind-target.png" width="400"/> <br/>
10. Drag from the event pin on the bind event node and create a custom event <br/><br/> <img src="/images/unreal/custom-event.png" width="400"/> <br/>
11. Finally, do something with the result. Here we just print to console. For Insert Data, the result will be a confirmation string. <br/><br/> <img src="/images/unreal/print-result.png" width="400"/>
